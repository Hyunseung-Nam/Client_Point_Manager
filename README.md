# CRM

프로토타입 전 단계의 내부용 CRM 데스크톱 앱입니다.  

## 상태
- 프로토타입 전 (미완성)
- 요구사항 검증 및 UX 정리 진행 중

## 개요
- PySide6 기반 데스크톱 앱
- 고객 목록 + 상세 패널 중심의 단순 입력 흐름
- 로컬 JSON 저장소 기반

## 기능 목록
- 고객 검색/선택 및 상세 정보 표시
- 원클릭 방문 기록 입력
- 자동 포인트 지급 및 사용 기록
- 최근 기록/전체 기록 조회
- 관리자 월간 통계 대시보드
- 고객 삭제(관리자 화면)

## 로드맵
- 입력 흐름 최적화(키보드/포커스/탭 순서 개선)
- 통계 지표 고도화 및 내보내기 옵션 검토
- 운영 로그/데이터 복구 흐름 강화

## 실행
```bash
python src/main.py
```

## 데이터 위치
- 개발 실행: 프로젝트 루트의 `data/`  
- 배포 실행: `AppData\Local\CRM\CRM\data\`

## 참고
> 이 저장소에는 실제 고객 데이터가 포함되지 않습니다.
